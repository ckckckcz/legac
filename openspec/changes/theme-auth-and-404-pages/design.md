# Design: Theme Auth and 404 Pages

## Visual Style
- **Typography**: Plus Jakarta Sans for headers, Geist Mono for code snippets.
- **Colors**:
  - Primary: `#34558b` (Brand Blue)
  - Background: White / `oklch(1 0 0)`
  - Accents: `bg-accent/50`, `bg-primary/10`
- **Surface**: Use cards with `border-border` and `rounded-2xl`.

## Screen Layouts
1. **404**: Minimalist, high-impact typography.
2. **Auth**: Split screen (on large screens) with a brand image and a clean form on white background.
